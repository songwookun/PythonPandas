# 2025 유저 동향 분석 보고서 
2025년 업데이트별 유저 반응을 시장(국내/해외) 및 출처(디시, 공식포럼, 카페) 기준으로 분석한 포트폴리오 자료를 포함하고 있습니다.  
업데이트 이후 1~2주간의 커뮤니티 반응을 정량화하고, 부정률·게시글 추이·부정 유형 등을 기반으로 핵심 인사이트를 도출

## 분석 개요
- 데이터 출처: 디시 갤러리 / 공식 포럼 / 네이버 카페  
- 데이터 구조: 전처리 → 정제 → 감정 라벨링(긍/부정) → 시장별 집계  
- 분석 관점  
  - 기간별 전체 게시글 수 변화  
  - 기간별 전체 게시글 부정률 및 지역(국내/해외) 비교  
  - 출처별 영향력  
  - 부정 반응 주요 유형  

## 기술 스택
- **Python**: pandas, numpy, sqlite3, matplotlib
- **DB**: SQLite
- **분석 환경**: Jupyter Notebook

## 제공 파일 구성

본 프로젝트는 총 5개의 핵심 파일로 구성

### 1. 문서·보고서
- **업데트별 유저동향_보고서.pdf**  
  - 유저 동향 분석의 핵심 인사이트와 그래프를 정리한 최종 보고서

### 2. 분석 코드
- **Data_Analysis.ipynb**  
  - SQLite DB의 뷰를 기반으로 전체 데이터를 처리·집계·시각화하는 분석 노트북

### 3. 데이터 파일
- **2025_유저_동향_자료.xlsx**  
  - 전처리된 요약 데이터(기간/출처/국가 기준 집계 포함)

### 4. 데이터베이스(DB)
- **user_feedback_normalized.db**  
  - 정규화된 SQLite 분석용 DB  

### 5. 데이터 생성 스크립트
- **Create_user_feedback_db.py**  
  - 원본 데이터를 SQLite DB 구조로 변환하는 스크립트
